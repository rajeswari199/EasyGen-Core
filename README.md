
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run build
$ npm run start:prod
```
## API Documentation
```bash
$ Open http://localhost:5001/api/ in browser for API documentation
```

## Docker Image

```bash
$ docker build -t easy-gen .
```
